IP routing domain instance.

It provides data plane isolation of routed traffic and control plane isolation of routing entities from other routing instances.

It allows for further fragmentation of routing domain.

Maximum number of Router MOs in node is limited to:
- For Baseband in RAN Compute Group 1, 2, or 3: 8,
- For Baseband in RAN Compute Group 4 or higher: 60.

IPv6 default hop limit.

Default value inserted into hop limit field of IPv6 header of datagrams, originated from all Router MOs, whenever hop limit value is not supplied by transport layer protocol.

Changed value takes effect on existing GPRS Tunneling Protocol paths and UDP sessions after node restart.

Disturbances: Changing this attribute too low might cause total loss of traffic or loss of connectivity to the node.
