Lldp

Represents Link Layer Discovery Protocol (LLDP) operation in Baseband.

When MO is created, LLDP Rx functionality is enabled on all EthernetPorts. Lldp MO must be created before LLDP Tx functionality can be enabled. LLPD Tx is enabled separately per EthernetPort MO.
