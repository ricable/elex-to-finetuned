class MacSecCp

ManagedElement
Transport
MacSecCp

Represents Controlled Port that can exist regardless if underlying hardware supports MACsec.

Only created when RiLink MO is created with following conditions:
- When one MacSecProfile MO is associated with RiLink MO.
- When Ethernet is detected on link.

Specification: IEEE 802.1AE 2018



This MO is created by the system.
