derivedDataType CurveList

Derived string for specification of elliptic curve list.

Contains colon-delimited (':') elliptic curve strings. Each elliptic curve string indicates individual elliptic curve. Character '!' means that preceded elliptic curve is permanently deleted from list. In case of negative expressions, 'ALL' needs to be used at the beginning.
For positive expressions, colon-delimited (':') elliptic curve strings are used.

Cipher strings with special meaning:
ALL - All elliptic curves listed in supportedEllipticCurves.

Examples:

ALL:!secp160r2
ALL:!secp160r2:!secp160r1:!secp160k1
secp160r2:secp160r1, assuming that listed elliptic curve strings are valid values of EllipticCurve derived data type.
