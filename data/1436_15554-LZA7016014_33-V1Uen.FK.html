MacSecSecurityLevel
minimumSecurityLevel = INTEGRITY_AND_CONFIDENTIALITY
Configures minimum required MACsec level.

If set to NONE, best-effort MACsec is provided.
If set to INTEGRITY_AND_CONFIDENTIALITY, both integrity and confidentiality protection are provided.
If available hardware does not allow both integrity and confidentiality, alarm is raised.

Disturbances: Changing this attribute can cause loss of traffic.
boolean
replayProtect = true
Value of replay protection.

If true, replay protection is provided.

Can only be set to false if no packet reordering is expected on link.
