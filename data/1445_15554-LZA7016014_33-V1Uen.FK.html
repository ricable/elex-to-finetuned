Attributes
boolean
fixedPosition = true
Whether node has fixed position.

If node does not move, this attribute is set to true. GNSS receiver may lock its position to enhance receiver performance.
If node may move, for example on cruise ship, this attribute is set to false. GNSS receiver needs to continuously track its position.

Regional telecom standard option to I, II, or III for ETSI (European Standard), ANSI (US Standard) or TTC (Japanese Standard) respectively.

Dependencies: If this attribute is changed, RadioEquipmentClockReference::adminQualityLevel, AssistingReference::remappedQualityLevel and RadioEquipmentClock::minQualityLevel attributes are changed to quality level value defined within telecom option.
